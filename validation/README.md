# 회원가입 validation
## 최종html = vaildation_3.html

1. 가입완료 버튼을 누를시 각 항목 체크
2. 아이디 (6~12 글자, 한글입력시 경고창)
3. 이메일 한글입력시 경고창
4. 비밀번호체크 (비밀번호 다를시 경고창)
5. 성별체크란에 하나라도 체크 안할 시 경고창
6. 이용약관 동의 안할시 경고창
7. 가입완료 버튼을 눌렀을때 확인하지 않은 사항 체크
8. 모든사항을 입력했을시 "가입이 완료되었습니다" 노출
